## Philosophy

Each skill in this repository:
- ✅ Is tested against the actual tool/service
- ✅ Documents real parameter names and formats
- ✅ Includes CLI vs MCP differences
- ✅ Has troubleshooting guides based on real errors

## Usage

```bash
# Clone to your skills directory
git clone git@github.com:WalterSumbon/my-skills.git
cd my-skills
./install-skills.sh your-skills-directory
```
